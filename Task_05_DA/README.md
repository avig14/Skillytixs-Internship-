# Task 5: Supermarket Sales Interactive Dashboard

## 📊 Project Overview
This repository contains the submission for **Task 5** of the Data Analytics Internship. [cite_start]The objective was to design a professional, interactive business dashboard to enable stakeholders to make data-driven decisions[cite: 4].

[cite_start]The dashboard focuses on usability, visual clarity, and storytelling with data to analyze supermarket sales trends, customer preferences, and branch performance[cite: 5].

## 📂 Dataset
* **Source:** Supermarket Sales 2.csv
* **Time Period:** January 1, 2024 - December 31, 2024
* **Key Columns:** Branch, Customer Type, Product Line, Unit Price, Quantity, Rating, Payment Method.
* **Data Processing:** A `Total Revenue` column was calculated (`Unit Price` * `Quantity`) during the data cleaning phase to allow for financial KPI analysis.

## 🛠️ Tools Used
* **Data Cleaning:** Python (Pandas) / Excel
* **Visualization:** Power BI / Tableau
* **Version Control:** GitHub

## 📈 Key Performance Indicators (KPIs)
[cite_start]Based on the analysis requirements, the following KPIs were visualized[cite: 9]:
1.  **Total Revenue:** The sum of all sales revenue generated.
2.  **Total Transactions:** The total count of sales invoices.
3.  **Average Rating:** Customer satisfaction score (scale 1-10).
4.  **Total Quantity Sold:** The total number of units sold across all categories.

## 🔍 Key Insights & Visuals
[cite_start]The dashboard includes the following visual elements to tell a data-driven story[cite: 5]:
* **Sales Trend Analysis:** A line chart tracking revenue over 2024 to identify seasonal peaks.
* **Branch Performance:** A comparison of sales between Brooklyn, Manhattan, and Queens branches.
* **Product Line Analysis:** Identifying high-performing categories (e.g., Food & Beverages vs. Electronics).
* **Payment Preferences:** A breakdown of usage between E-wallets, Cash, and Credit Cards.
* [cite_start]**Slicers/Filters:** Interactive filters for Branch, Customer Type, and Date were added for user-centric exploration[cite: 10].

## 🚀 How to Use
1.  Clone this repository.
2.  Open the `.html` dashboard file (or `.pbix` if using Power BI).
3.  Use the dropdown slicers at the top to filter data by Branch or Customer Type.
